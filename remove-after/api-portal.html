Authentication

Use a bearer token. Always send an Idempotency-Key for any request that can be retried.

zntl_live_xxxxxxxxxxxxxxxxxxxxx 
curl -s https://api.zentla.dev/v1/health \
  -H "Authorization: Bearer zntl_live_xxxxxxxxxxxxxxxxxxxxx"
